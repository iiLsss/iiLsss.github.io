二、SVN与Git的最主要的区别？

Git是分布式版本控制系统。

git本地操作

GIT版本库有暂存区和版本区

  • 工作区：在工作区对文件进行操作
  • 暂存区：git add将添加或修改文件加入暂存区
  • 版本区：git commit把暂存区的所有内容提交到版本区

初始化git

  • 不能再文件夹里初始化后，在进去子文件夹初始化。不能嵌套

取消初始化

  • 删除文件夹中自动生成.git 隐藏文件即可

git的对比

  • git diff 工作区和暂存区
  • git diff <分支名> 工作区和历史区
  • git diff –cached 暂存区和历史区比较
